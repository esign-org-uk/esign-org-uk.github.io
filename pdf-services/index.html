Getting Started

  • Head to the PDF Services API page, and:
    • Obtain credentials that will allow you to authenticate successfully
    • Download the sample Postman collection, and load into your environment
    • Adjust your Postman environment settings, to ensure your credentials are incorporated
    • Work your way through the list of PDF OPERATIONS, exploring the available options as you go
  • To assist with the Document Generation operation, here are some resources for you to start with (or enhance as you wish):
    • A Microsoft Word document that contains a sample contract, sample-contract.docx
    • A JSON file that contains the raw data for the same candidate, to be used to tag & test document generation
    • A JSON file that contains the raw data for a different candidate, to test your tagging accuracy
    • Use as many of the document generation capabilities, such as conditional logic, to ensure your template caters for missing JSON data, etc.
