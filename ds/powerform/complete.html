DocuSign PowerForm Download

Thank you for providing your consent. You can now follow the steps below to extract your DocuSign PowerForm definitions.

Download the Java-based tool to your local machine:

  • docusign-powerform-download

Open a command prompt and issue the following command (replacing the angle-bracketed values as appropriate):

            
                java -jar docusign-powerform-download-<version>.jar <accountId> <userId> <outputDirectory>

Assuming that you have entered the correct details, you should see output similar to:

            
                Successfully authenticated to DocuSign
                ======================================
                <n> PowerForm(s) found in accountId ...
                PowerForm '<powerFormName>' written to <powerFormId>.json
