DocuSign PowerForm Download

By default, the DocuSign web interface does not provide any mechanism to download PowerForm definitions.

In order to overcome this limitation a command-line tool has been developed, utilising the DocuSign API, to allow for download of JSON-formatted definitions to the local filesystem.

As the tool makes use of the DocuSign API, it is necessary for an account administrator to provide consent for the tool to impersonate the user to extract PowerForm definitions.

To provide this consent, please click on the link below and follow the OAuth process to permit access.
